# 굴러굴러 던전 (Rolling Dungeon)
**장르:** 1인 자동전투 로그라이트 (Solo Auto-battler Roguelite)

이 게임은 플레이어가 캐릭터를 직접 조작하지 않고, 매 라운드 전투 결과를 바탕으로 업그레이드를 선택하여 캐릭터를 끊임없이 성장시키는 **1인 자동전투 로그라이트 게임**입니다.

---

## 1. 게임의 핵심 흐름 (Game Loop)

1. **Home → Loadout Select**: 영구적으로 잠금 해제한 무기 중 시작 무기 1개를 선택하고, 장신구/신발/갑옷 슬롯 장비를 확인합니다.
2. **Run Loadout Lock-in**: 입장 시점의 시작 무기, 장착 장비, 해금 무기 목록이 이번 런의 빌드 기반으로 고정됩니다.
3. **Stage Start**: 현재 스테이지 목표와 적 정보가 간략히 표시되며 전투에 진입합니다.
4. **Auto Battle (자동 전투)**: 플레이어 캐릭터는 육각형 경기장 안에서 AI 적들과 자동으로 전투를 벌입니다. 캐릭터는 당구공처럼 벽에 튕기며 이동하고, 충돌이나 무기 스킬을 이용해 적에게 피해를 줍니다.
5. **Stage Clear / Defeat**:
    - **Clear**: 라운드의 모든 적(또는 보스)을 처치하면 스테이지를 클리어하고 골드와 보상을 얻습니다.
    - **Defeat**: 적의 공격에 체력이 0이 되면 남은 목숨(Life)이 감소하며, 작은 패배 보정을 받고 해당 스테이지를 재도전합니다. 목숨이 0이 되면 런 종료.
6. **Upgrade Select / Run Armory**: 스테이지 클리어 후 3개의 무작위 증강 중 하나를 선택합니다. 장착한 손/신발/갑옷은 증강 후보로 등장해 투사체 수, 쿨타임, 이동, 방어 수치 등을 발전시킬 수 있습니다.
7. **Next Stage**: 다음 스테이지로 진입하며 적들이 점점 강해집니다.
8. **무한 사이클 (Infinite Cycles)**: 매 사이클은 3개의 일반 스테이지 + 1개의 보스 스테이지로 구성됩니다. 보스를 클리어하면 다음 사이클로 진입하며, 적의 체력/공격력/속도가 사이클에 비례하여 증가합니다. 끝이 없으며 플레이어가 사망할 때까지 계속됩니다.
9. **Run Results (런 결과)**: 사망 시 런 통계(도달 스테이지, 처치 수, 피해량 등)와 달성한 업적이 표시되고, 업적 보상으로 크리스탈(영구 재화)을 획득합니다.

---

## 2. 전투 시스템

### 자동 조작과 물리 엔진
- **이동:** 캐릭터는 마치 당구공처럼 특정 속도를 가지고 경기장 안을 통통 튀어 다닙니다. 방향키로 직접 조작할 필요가 없습니다.
- **충돌 피해:** 적과 부딪히면 서로 밀려나며 피해를 주고받습니다. (두꺼운 외곽선의 카툰/스케치풍 이펙트)

### 스탯과 자원
- **HP (체력):** 공격을 받으면 감소하며, 0이 되면 목숨을 1개 잃습니다.
- **ATK (공격력):** 부딪히거나 기본 무기로 적을 공격할 때 입히는 피해량입니다.
- **DEF (방어력):** 적에게서 받는 피해를 줄여줍니다.
- **SPD (이동 속도):** 캐릭터가 이동하는 속도이며, 스킬의 쿨다운을 줄이는 데에도 영향을 미칩니다.
- **골드 (Gold):** 경기장에 스폰되는 골드 칩을 먹거나, 적을 처치할 때 획득하며 인런 상점에서 사용됩니다.

---

## 3. 시작 무기와 런 중 장착

게임 시작 시 해금된 무기 중 한 가지를 시작 무기로 선택합니다:
- **Gunner (거너):** 근처 적에게 총알을 지속적으로 발사. (기본 공격력 낮음, 공격 속도 빠름)
- **Blade (검사):** 일정 주기마다 주변 적을 넓게 베어버림. (범위 공격, 기본 체력 약간 높음)
- **Miner (광부):** 이동 경로에 지뢰를 깔아 밟은 적에게 큰 피해. (전술적 이동, 이동 속도 빠름)
- **Laser (레이저):** 가장 가까운 적에게 강력하고 지속적인 레이저 빔. (단일 개체 극딜)

입장 전 선택은 빌드의 방향을 정하는 역할만 합니다. 스테이지를 진행하며 열리는 런 장착 상점에는 이번 런 시작 시점에 영구 해금되어 있던 무기만 등장하고, 이미 시작 무기로 들고 있거나 이번 런에서 장착한 무기는 다시 등장하지 않습니다.

### 장비 슬롯
- **무기:** 공격 방식 자체를 결정합니다. 시작 무기로 1개를 들고 들어가며, 런 중 해금 무기를 추가 장착할 수 있습니다.
- **장신구/손:** 공격 보조, 투사체 수, 반사, 치명타 감각을 담당합니다.
- **신발/발:** 이동 속도, 회피, 충돌 템포, 쿨타임 감소를 담당합니다.
- **갑옷/옷:** 체력, 방어, 피격 전 보호막을 담당합니다.

## 4. 무한 사이클 & 적 구성

### 사이클 구조
- 매 사이클은 **3개의 일반 스테이지** + **1개의 보스 스테이지**로 구성됩니다.
- 사이클이 올라갈수록 적의 모든 스탯이 강화됩니다:
  - **HP**: `기본 × (1 + 사이클 × 0.35)`
  - **ATK**: `기본 × (1 + 사이클 × 0.25)`
  - **SPD**: `기본 × (1 + 사이클 × 0.05)`

### 적의 종류
- **Basic (기본):** 평범한 스탯.
- **Fast (빠른 적):** 체력은 낮지만 속도가 빠름.
- **Dasher (돌진):** 플레이어를 향해 돌진.
- **Shooter (사격):** 원거리에서 탄환 발사.
- **Tank (탱커):** 크기가 크고 방어력이 높으나 느림.
- **Shield (방패):** 일정 시간 피해 감소.
- **Splitter (분열):** 사망 시 작은 적으로 분열.
- **Bruiser (브루저):** 높은 공격력과 충격 효과.

### 보스 패턴 (5사이클 주기 반복)
1. **기본 보스** — 거대한 크기, 방패 능력
2. **소환 보스** — 작은 미니언 소환
3. **돌진 보스** — 강력한 돌진 공격
4. **방패 보스** — 방어 페이즈 전환
5. **복합 보스** — 여러 패턴 조합

## 5. 업그레이드 시스템 (인런 로그라이크)

매 스테이지 클리어 후 3가지 카드 중 하나를 선택합니다. (등급: 일반 / 희귀 / 에픽)
- **Assault (광폭화):** 공격력 대폭 상승.
- **Guard (철벽):** 방어력/최대 체력 상승.
- **Body Small (소형화):** 크기 감소, 회피력 상승.
- **Body Big (대형화):** 크기 증가, 체력 보너스.
- **Weapon Count:** 추가 무기 슬롯.
- **Bullet Burst:** 발사체 수 증가.
- **Bullet Reflect:** 반사 횟수 증가.
- **Barrier:** 재생 방벽 획득.

증강은 기본 스탯과 장착 장비를 발전시키는 축이고, 런 장착 상점은 해금한 무기를 추가로 붙여 빌드를 넓히는 축입니다. 해금하지 않은 무기는 런 중에도 등장하지 않습니다.

## 6. 메타 진행 시스템 (Meta Progression)

### 크리스탈 (영구 재화)
- 크리스탈은 **업적 달성 시에만** 획득됩니다.
- 적을 처치하거나 런 중에 얻는 것이 아닙니다.
- 크리스탈로 메타 상점에서 영구 무기를 구매합니다.

### 업적 시스템
- **킬 업적**: 누적 처치 수에 따라 보상 (10/50/100/200마리)
- **스테이지 업적**: 최고 도달 스테이지에 따라 보상 (5/10/20/30)
- **보스 업적**: 누적 보스 처치 수에 따라 보상 (1/3/5/10)
- **피해 업적**: 누적 총 피해량에 따라 보상 (1,000/5,000/20,000)

### 메타 상점
- 크리스탈로 무기를 영구적으로 잠금 해제합니다.
- 해제된 무기는 런 시작 전 시작 무기로 선택할 수 있고, 런 중 장착 후보로도 등장합니다.
- 장비 탭에서 장신구/손, 신발/발, 갑옷/옷 슬롯 장비를 구매하고 각 슬롯에 1개씩 장착합니다.
- 기본 무기(Gunner)는 항상 사용 가능합니다.
- 구매 가능 무기: 미니건, 장거리 포, 독 가스, 회전 칼날, 지뢰, 불타는 발자국, 전방위 버스트, 거대 대검, 도탄 사격, 수호자의 오라

## 7. 게임 플로우 요약

```
홈 화면 (시작 / 상점)
  → 시작 무기 선택 + 장비 슬롯 확인
      → 자동 전투 (무한 사이클)
        → 스테이지 클리어 → 증강 선택 / 장비 발전 / 해금 무기 추가 장착 → 다음 스테이지
        → 보스 클리어 → 업그레이드 선택 → 다음 사이클 (더 강한 적)
        → 사망 → 런 결과 화면
          → 업적 확인 → 크리스탈 획득
          → 홈 화면으로 복귀

홈 화면 → 상점
  → 메타 상점
    → 크리스탈로 무기 구매
    → 업적 진행도 확인
```

## 8. 시각적 정체성 (Art Style)
- **스케치 & 카툰 (Sketch & Cartoon):** 종이 같은 배경 위에서 펼쳐지는 낙서 같은 전장.
- 굵고 검은 잉크 외곽선(Ink Outlines).
- 뚜렷한 원색(Red, Blue, Yellow, Green) 기반의 고대비 색상.
- 큼직하고 타격감 넘치는 데미지 폰트 출력.
