# 증강 시스템 정리

## 진행 방식

- 스테이지를 클리어하면 `증강 선택` 화면으로 이동한다.
- 매번 전체 증강 목록 중 3장이 랜덤으로 제시된다.
- 1장을 선택하면 즉시 효과가 적용되고 다음 스테이지로 진행한다.
- 같은 화면의 런 장착 상점에는 이번 런 시작 시점에 영구 해금된 무기만 등장한다.
- 시작 무기와 이미 런 중 장착한 무기는 장착 후보에서 제외된다.
- 입장 전 장착한 손/신발/갑옷 슬롯이 있으면 해당 슬롯을 발전시키는 증강도 후보에 포함된다.
- 증강 희귀도는 `COMMON`, `RARE`, `EPIC` 3단계다.
- 희귀도가 높을수록 같은 증강이라도 적용 수치가 커진다.

## 희귀도 배율

| 희귀도 | 확률 | 배율 |
| --- | ---: | ---: |
| COMMON | 65% | 1.0x |
| RARE | 25% | 1.6x |
| EPIC | 10% | 2.5x |

## 스탯 용어

| 표기 | 의미 |
| --- | --- |
| HP | 최대 체력 |
| ATK | 공격력 |
| DEF | 방어력 |
| SPD | 이동 속도. 높을수록 일부 스킬 쿨타임도 짧아진다. |
| PWR | 무기 위력 계수 |
| WPN | 무기 형태 레벨. 무기의 크기, 범위, 개수, 지속시간을 강화한다. |
| SHD | 보호막. 체력보다 먼저 피해를 흡수한다. |
| REG | 초당 체력 재생량 |
| LIFE | 흡혈률. 준 피해 일부만큼 체력을 회복한다. |

## 증강 목록

| 증강 | 주요 효과 | 실제 적용 |
| --- | --- | --- |
| 공격 강화 | ATK 증가 | 모든 기본 충돌/무기 피해가 증가한다. |
| 방어 강화 | DEF 증가, HP 소폭 증가 | 받는 피해가 감소하고 최대 체력이 오른다. |
| 속도 강화 | SPD 증가 | 이동 속도가 오르고 스킬 쿨타임이 감소한다. |
| 생존 본능 | 최대 HP 증가, 즉시 회복 | 최대 체력이 오르고 현재 체력도 회복된다. |
| 무기 강화 | ATK 증가, PWR 증가 | 현재 캐릭터 무기의 피해 계수가 오른다. |
| 무기 개조 | WPN 증가 | 무기 형태가 발전한다. 무기별로 강화되는 항목이 다르다. |
| 보호막 | SHD 증가 | 다음 전투부터 보호막이 먼저 피해를 흡수한다. |
| 손 장비 개조 | ATK, 투사체 수, 반사 증가 | 손/장신구 슬롯 장비가 있을 때만 등장한다. |
| 신발 개조 | SPD 증가, 크기 소폭 감소 | 이동 속도가 올라 일부 무기 쿨타임도 짧아진다. |
| 갑옷 개조 | HP, DEF, 보호막 증가 | 갑옷/옷 슬롯 장비가 있을 때만 등장한다. |
| 자가 회복 | REG 증가 | 전투 중 매초 체력을 회복한다. |
| 생명 흡수 | LIFE 증가 | 적에게 실제 체력 피해를 주면 일부를 회복한다. |
| 과부하 | ATK 증가, SPD 증가 | 공격과 기동력을 동시에 올린다. |
| 방호 장갑 | DEF 증가, SHD 증가 | 방어력과 보호막을 함께 올린다. |

## 무기 개조 효과

`무기 개조`는 현재 선택한 캐릭터 무기에 따라 효과가 다르게 나타난다.

| 무기 | WPN 증가 효과 |
| --- | --- |
| 총 | 탄환 크기 증가, 추가 탄환, 추가 반사, 탄 퍼짐 증가 |
| 칼 | 회전 칼 범위 증가, 접촉 판정 폭 증가, 접촉 피해 증가 |
| 독 | 독 구름 반경 증가, 지속시간 증가, 초당 피해 증가 |
| 지뢰 | 지뢰 반경 증가, 일정 레벨마다 설치 개수 증가, 폭발 피해 증가 |
| 레이저 | 레이저 폭 증가, 사거리 증가, 피해 증가 |

## 현재 전투 흐름

1. 해금된 무기 중 시작 무기 1개를 선택한다.
2. 상점에서 해금한 손/신발/갑옷 장비를 슬롯별로 장착한다.
3. 스테이지 전투가 시작된다.
4. 승리하면 증강 3개 중 하나를 고른다.
5. 장착한 슬롯 장비가 있으면 해당 장비를 발전시키는 증강이 후보에 섞인다.
6. 골드가 충분하고 해금된 후보가 있으면 무기를 추가 장착할 수 있다.
7. 선택한 증강과 추가 장착 무기는 이번 런 동안 누적된다.
8. 다음 스테이지에서 누적된 스탯과 특수 효과로 전투한다.
9. 런은 무한 사이클로 진행되며 사망하면 종료된다.

## 밸런스 메모

- 보호막은 체력보다 먼저 소모되며, 다음 스테이지 진입 시 최대 보호막까지 회복된다.
- 재생과 흡혈은 생존력을 크게 올리므로 높은 희귀도에서 체감이 크다.
- 무기 개조는 단순 피해 증가가 아니라 무기의 형태와 판정 자체를 키우는 성장 축이다.
- `무기 강화(PWR)`은 피해 계수 강화, `무기 개조(WPN)`은 형태 강화로 역할을 분리한다.
